# Shopify Sales Analysis

This project analyzes Shopify eCommerce sales data using R and SQL.

## Goals

* Analyze seasonal sales trends
* Identify best-selling swimwear products
* Compare returning vs new customers
* Explore customer retention patterns
* Build business-oriented dashboards

## Tools

* R
* SQL
* Shopify CSV exports
* ggplot2
* dplyr

## Planned Analysis

* Best Selling Bikinis by Season
* Revenue Trends
* Customer Retention
* Conversion Funnel
* Cart Abandonment
* Revenue Forecasting

Work in progress.
